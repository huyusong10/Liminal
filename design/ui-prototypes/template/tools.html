⚙ Orchestrations ◧ Roles 🛠 Tools

Tools

📁
file_read
Read the contents of a file in the workspace. Supports text files, code files, and markdown.
✏️
file_write
Write or overwrite a file in the workspace. Creates parent directories if needed.
🗑️
file_delete
Delete a file or directory from the workspace. Use with caution.
🔍
file_search
Search for files in the workspace by name pattern or content.
💻
shell
Execute shell commands in the workspace. Output is captured and returned.
🌐
web_search
Search the web for information. Returns search results with snippets.
📊
web_fetch
Fetch the content of a web page. Returns the page text content.
🧪
test_run
Run tests in the workspace. Supports various test frameworks.
📦
git_status
Check the git status of the workspace. Shows modified and untracked files.
🔀
git_diff
Show the git diff of changes in the workspace. Useful for reviewing modifications.
⏱️
sleep
Pause execution for a specified duration. Useful for rate limiting.
📝
ask_user
Ask the user a question and wait for a response. Use for human-in-the-loop decisions.
