# 场景：创建循环时借助就近入口回看角色与编排

**目标**：确认“创建循环”页把主表单和辅助捷径放在同一工作台头部时，用户仍能清楚分辨主任务，并顺畅往返角色定义、流程编排和教程。

**前置**：系统里至少存在一个可选编排；用户准备新建一个 loop，但还想先确认角色与流程设置。

**步骤**：
1. 打开“创建循环”页，先浏览工作台头部的说明文案和右侧快捷入口，确认主表单仍以“选择编排方案”为中心。
2. 用键盘或读屏聚焦字段旁的 tips 按钮，确认这些帮助入口不会只读出一个 `i`，而是会提示当前语言下的说明语义。
3. 点击“角色定义”，确认能进入角色列表查看默认执行配置，然后返回创建页。
4. 点击“流程编排”，打开任意一个编排查看步骤顺序，再返回创建页。
5. 点击“使用教程”，确认能快速判断这次改动应该回到角色、编排还是 loop 层，再返回创建页。
6. 回到创建页后，继续选择编排、填写 workdir 与 spec 路径，并确认之前的操作没有把主表单挤到次要位置。

**预期**：工作台头部里的快捷入口始终只是辅助路径；tips 按钮对辅助技术会读出当前语言的帮助语义；无论往返多少次，创建 loop 的主任务、当前表单层级和应填写的核心字段都保持清晰。
