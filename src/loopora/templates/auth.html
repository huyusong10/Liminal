需要访问令牌 / Auth token required

这个 Loopora 实例正在网络模式下运行，所以先带上令牌再进来比较稳妥。

This Loopora instance is exposed over the network, so it expects an auth token before letting requests through.

把令牌加到地址后面访问一次即可，例如：{{ url_path }}?token=<your-token>

You can also send it as Authorization: Bearer <your-token> or X-Loopora-Token.
