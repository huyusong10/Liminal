需要访问令牌 Auth token required

这个 Loopora 实例正在网络模式下运行，所以先带上令牌再进来比较稳妥。 This Loopora instance is exposed over the network, so it expects an auth token before letting requests through.

把令牌加到地址后面访问一次即可，例如：{{ url_path }}?token=<your-token> Add the token to the URL once, for example: {{ url_path }}?token=<your-token>

你也可以通过请求头发送：Authorization: Bearer <your-token>、X-Loopora-Token，或兼容旧实例的 X-Liminal-Token。 You can also send it via headers: Authorization: Bearer <your-token>, X-Loopora-Token, or the legacy-compatible X-Liminal-Token.
