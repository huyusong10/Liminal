当强 Agent 能做，但人类不该反复回来确认时，再用 Loopora。 Use Loopora when a strong agent could do the work, but humans should not have to keep coming back to confirm and redirect it.

Loopora 不是因为 agent 完全做不到，而是因为如果没有一条能承载协作姿态的流程，人类就得在每个关键阶段反复回来检查、裁决和纠偏。现在推荐先让外部 Agent + Skill 对齐任务，产出 bundle，再在“创建循环”页导入运行。 Loopora is not here because the agent is helpless. It is here because, without a workflow that carries the right collaboration posture, humans keep getting pulled back in to inspect, judge, and redirect the work after every meaningful step. The recommended path is now to align with an external Agent + Skill, produce a bundle, then import it from Create Loop.

核心精神Core idea

编译协作姿态Compile collaboration posture

Loopora 不要求你先写出一套完整规则。它更关心这次任务里的协作姿态：什么算真证据、什么是假完成、什么时候该重构、什么时候该先裁决再继续。 Loopora does not ask you to write a complete rulebook first. It cares about the task-scoped posture: what counts as evidence, what is fake done, when refactoring matters, and when judgment should happen before more work.

三层共同承载Carried by three surfaces

姿态不是一段 prompt。`spec` 定义成功和风险，角色定义表达每个角色的行为气质，workflow 决定这些判断何时发生。三者合在一起，才是可运行的协作约定。 Posture is not a single prompt. The `spec` defines success and risk, role definitions express how each role should behave, and the workflow decides when those judgments happen. Together they absorb the supervision work that would otherwise keep dragging humans back in.

Bundle 是入口Bundle is the entry

working agreement 只在对齐阶段存在，用来让用户确认“Loopora 是否理解了我”。最终导入的是单文件 YAML bundle；它既能被人读懂，也能直接物化成 loop、角色定义、spec 和流程编排。 The working agreement is transient: it helps the user confirm whether Loopora understood the task. The durable artifact is a single YAML bundle that is readable by humans and runnable as a loop, roles, spec, and orchestration.

什么时候别用When not to use it

如果一位强 Agent 加上一轮人工 review 就够了，就别开 loop。Loopora 只该用在那些本来会把人类反复拉回来确认和纠偏，而且后续还需要用模糊反馈继续修订姿态的长程任务上。 If one strong agent pass plus one human review is enough, skip the loop. Use Loopora only for long tasks that would otherwise keep pulling humans back in to confirm, redirect, and later revise the posture with fuzzy feedback.
