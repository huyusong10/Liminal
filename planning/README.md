# Loopora Planning Notes

This directory keeps roadmap and phase-tracking notes that are useful for product
direction but are not stable module contracts.

- Keep current design contracts in `design/core-ideas/` and `design/detailed-design/`.
- Do not require routine code changes to update planning notes.
- Move a planning note into `design/` only when it defines a stable module boundary,
  public contract, lifecycle, dependency direction, or invariant.
